Cloud by Example Networking Company

This is a possible setup for hosting the new Cloud Network of ENC.
Some services are there for basic routings and classifications.
This host also provides other core functionalities.

Firewall

firewall.example.com

The complete traffic should lead to this machine.
It is based on iptables which has two chains. In this case: filter and nat.
Also, there are some security features supported and activated.
SSH brute force Prevention ICMP flood Prevention port scan Prevention

Reverse Proxy

proxy.example.com

This reverse proxy is an instance of Nginx Proxy Manager
Every web traffic should point to this service.
Furthermore, it is managing SSL certificates for any domain.

DNS

dns.example.com

The DNS server is an instance of AdGuard Home
The main tasks of AdGuard are blocking annoying domains and
resolving the name of the requesters by their ip address.

OpenVPN

The OpenVPN servers's IP address is 1.2.3.4 alias cloud.example.vpn
To connect to the VPN just get the OpenVPN Client and
download the following profile: client.example.vpn.ovpn

Docker

cloud.example.vpn:9000

This server makes use of Docker and
uses a suitable management interface called Portainer
For example, the reverse proxy is running as a docker container.

Facts

This is a VM hosted by Example Provider which only needs
1 shared processor core, 1 GB of memory and 10 GB storage.
It costs about € 120 per year.
